Übersicht

Willkommen im Einsatzleitsystem (Command Center). Diese Software unterstützt Führungskräfte bei der Verwaltung von komplexen Einsatzlagen, der Zuweisung von Einsatzabschnitten (EA) und der Dokumentation.

Linke Spalte (Einsatzliste)

  • Zeigt alle aktiven Einsätze.
  • Such- und Filterfunktionen.
  • Zugriff auf das Archiv (abgeschlossene Einsätze).

Rechte Spalte (Lagekarte)

  • Interaktive Karte der Einsatzstelle.
  • Visuelle Darstellung von EAs (Farbzonen).
  • Pins zeigen genaue Standorte und Status an.

Einsätze anlegen

Drücken Sie den großen roten Button "NEUER EINSATZ" oben links, um das Alarmierungsfenster zu öffnen. Sie haben zwei Möglichkeiten, den Ort zu bestimmen:

Variante A: Automatische Suche

Geben Sie Straße und Hausnummer in das Adressfeld ein. Das System schlägt automatisch Adressen vor. Klicken Sie auf einen Vorschlag, um ihn zu übernehmen.

Variante B: Karte nutzen

Klicken Sie auf "AUF KARTE WÄHLEN". Klicken Sie anschließend direkt auf das betroffene Gebäude in der Karte. Das System ermittelt automatisch die nächstgelegene Adresse.

Hinweis zu Duplikaten: Wenn Sie einen Einsatz in unmittelbarer Nähe (15m) eines bestehenden Einsatzes anlegen, warnt Sie das System vor einem möglichen Duplikat.

Status & Disposition

Jeder Einsatz durchläuft drei Phasen. Sie können den Status ändern, indem Sie einen Einsatz in der Liste anklicken und die entsprechenden Buttons nutzen.

Phase 1

Offen

Einsatz ist angelegt, aber noch keine Kräfte alarmiert/eingetroffen. Roter Pin.

Phase 2

Disponiert

Einheiten sind alarmiert oder vor Ort. Maßnahmen laufen. Blauer Pin.

Phase 3

Abgeschlossen

Einsatz beendet. Einsatz wird ins Archiv verschoben. Orangener Pin.

Einsätze bearbeiten oder verschieben

Klicken Sie auf das kleine Stift-Symbol neben einem Einsatz in der Liste, um das Bearbeitungsfenster zu öffnen. Hier können Sie:

  • Den Standort des Markers auf der Karte korrigieren.
  • Das Alarmstichwort ändern.
  • Die Zuordnung zum Einsatzabschnitt (EA) manuell überschreiben.
  • Den Einsatz löschen.

Einsatzabschnitte (EA)

Das System ordnet Einsätze automatisch geografischen Zonen zu. Diese Zonen (EAs) müssen Sie zuerst zeichnen.

So erstellen Sie einen EA

1
Klicken Sie oben im Menü auf "EA VERWALTEN". Geben Sie Name (z.B. "EA West"), Leitung und Farbe ein. Klicken Sie auf "Erstellen".
2
Klicken Sie in der Liste auf das Stift-Symbol neben dem neuen EA. Der Zeichenmodus startet.
3
Klicken Sie auf die Karte, um die Eckpunkte der Zone zu setzen.
TIPP: Wenn Sie in die Nähe eines anderen EA klicken, "snappt" der Punkt automatisch an die Grenze, um Lücken zu vermeiden.
4
Klicken Sie erneut auf den Startpunkt, um die Fläche zu schließen und zu speichern.
Alle Einsätze innerhalb dieser Fläche werden nun automatisch diesem EA zugeordnet.

Bericht & Export

Am Ende einer Schicht oder Lage können Sie ein vollständiges Protokoll exportieren.

  1. Klicken Sie oben auf den Button "BERICHT / ENDE".
  2. Prüfen Sie die Statistik (Anzahl Einsätze, Datum).
  3. Tragen Sie den Namen des Protokollführers oder Lagezeichners ein.
  4. Klicken Sie auf "BERICHT HERUNTERLADEN".

Es wird eine HTML-Datei heruntergeladen, die Sie per E-Mail versenden oder archivieren können. Diese Datei enthält eine interaktive Karte und die komplette Historie und kann ohne Internet geöffnet werden.

Offline Modus

Was passiert bei Internetausfall?

Sollte die Internetverbindung abbrechen, schaltet das System automatisch in den Notfall-Lese-Modus.

Ein schwarzer Bildschirm überlagert die Anwendung. Sie können keine neuen Einsätze anlegen, haben aber weiterhin Lesezugriff auf alle bisherigen Daten, Adressen und EA-Zuweisungen, um den Betrieb manuell (per Papier/Funk) fortzuführen. Sobald das Internet zurückkehrt, verschwindet der Bildschirm automatisch.
